## Фазы

- [x] Убрана метрика «97% положительных отзывов» из hero.
- [x] Hero крупнее на больших экранах: заголовок до 5.4rem, текст до 20px, кнопки и метрики масштабируются (clamp).
- [x] Карточка «Состав»: добавлены значимые флаги — «Супер состав», «Рекомендовано», «Потенциальный аллерген» (янтарное предупреждение).
- [x] «Индивидуальный уход для волос» — новые 4 шага: Глубокое очищение (раз в неделю по показаниям) → Очищение (шампунь под тип кожи головы) → Уход (смываемые средства, красота и восстановление) → Защита (несмываемый финиш от среды).
- [x] «Схема ухода для лица»: Утро — Очищение → Активы → Увлажнение и барьер → Защита от солнца; Вечер — Демакияж → Очищение → Активы → Увлажнение и барьер.
- [x] **Змейка-прогресс** вместо батарейки: тело-синусоида, гонится за 10 яблочками, ест их и удлиняется; шелковистые локоны развеваются назад и растут с каждой неделей; подписи недель сохранены. Чистый rAF без setState на кадр (только смена недели).
- [x] Фон: только 2 цветка из спрайта (кошка вырезана попиксельно → flower-pink.png, flower-blue.png), цветов меньше (6), крупнее (до 860px), часть заблюрена; **уплывают от курсора** медленно, как по воде (lerp 0.018); +3 фиолетовых размытых пятна. Фон приложения — те же вырезанные цветы (4 шт.).
- [x] «Почему так происходит»: тексты переписаны в формате «боль → как решаем», без принижения других приложений; добавлен подзаголовок про «один сервис для всех бьюти-вопросов».
- [x] «Ваша схема ухода»: конкретные средства (пилинг кожи головы, шампунь под тип кожи, комплексная маска, поверхностный кондиционер, термозащита под рутину укладки, масло-финиш) + блок «Рутина для кудряшек».
- [x] «Три шага до результата»: профессиональный полный тест; анализ всего арсенала (волосы, лицо, декоративка); схема + дневник прогресса + движение к бьюти-целям вместе.
- [x] Отзывы: 6 карточек, только девушки; закрыты острые боли ЦА (уход не работал, окрашенные волосы-солома, перхоть, пушистость/электризация, хаотичный уход и траты, кожа лица).
- [x] Тарифы: бесплатный конкретизирован (7 поисков/нед, 3 анализа своих средств/нед, краткий тест, справочник, сравнение до 2) vs подписка (безлимит, полный тест + подбор, схемы волосы+лицо, совместимость, дневник прогресса, сравнение до 5 + аналоги, рутина для кудрявых). Те же списки — в онбординге регистрации.
- [x] Регистрация: маскот с двигающимися глазами (общий `MascotFigure` с экраном входа).
