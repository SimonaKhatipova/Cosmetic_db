# Поиск по алиасам во вкладке «Ингредиенты»

**Дата:** 2026-06-12
**Статус:** реализован (не закоммичен)

## Проблема

Поиск в каталоге ингредиентов (`src/App.jsx`, фильтр в `filtered`) смотрел только в `inci_name` и `ru_name`. Альтернативные написания канонического ингредиента лежат в `aliases` (через `|`) и в поиск не попадали: запрос «Fragrance» не находил канонический Parfum.

## Решение

- [x] В условие поиска добавлена проверка `(i.aliases || "").toLowerCase().includes(q)`. `aliases` уже грузится в `adaptIngredient`, дополнительных запросов нет.
- [x] Проверено: счётчики и сортировка работают по уже отфильтрованным строкам, причина матча на них не влияет.
- [x] Гейты: `npm run lint` — 54 замечания (базовый уровень, новых нет), `npm run build` ✓.
- [x] Подсветка: если строка нашлась только по алиасу — под INCI-именем показывается «найдено по алиасу: …» с конкретным алиасом (видно, почему «Fragrance» показал Parfum).

## Итог

**Реализован целиком**, включая подсветку алиасных совпадений.
