# Обзор продукта

**Название:** Beauty Helper
**Тип:** Веб-приложение + Telegram Mini App + мобильное приложение (iOS/Android)
**Текущий стек:** React (фронтенд), Supabase (БД), GitHub Pages (хостинг MVP)
**Ссылка на MVP:** https://simonakhatipova.github.io/Cosmetic_db/

## Что умеет приложение

- Анализ состава косметического средства по номенклатуре INCI
- Проверка безопасности ингредиентов
- Определение функций каждого компонента
- Подбор ухода под тип кожи / волос
- Поиск аналогов средства
- Сравнение составов двух и более средств
- Построение рабочей схемы ухода (лицо + волосы)
- В перспективе: скан этикетки камерой → персональная рекомендация

## Категории косметики

Любые средства по INCI-номенклатуре:
- Уход за волосами (шампуни, маски, бальзамы, масла)
- Уход за кожей лица (кремы, сыворотки, тоники)
- Тело (лосьоны, скрабы, масла)
- Макияж (тушь, тональные средства, помады)

## Ключевая ценность

Помочь человеку не тратить деньги на пустые банки — подобрать реально работающий уход и понять, что он наносит на кожу.
